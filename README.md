# Service Booking

A simple cross-platform Flutter service booking app for Android and iOS with service cards, booking form, and appointment list.

## Features

- Clean Flutter Material 3 user interface
- Android and iOS compatible Flutter structure
- Simple state management using StatefulWidget
- Beginner-friendly code organization
- Ready for GitHub portfolio upload

## Technologies Used

- Flutter
- Dart
- Material Design

## Getting Started

1. Install Flutter SDK.
2. Open this project folder in VS Code or Android Studio.
3. Run:

```bash
flutter pub get
flutter run
```

## Build Commands

Android APK:

```bash
flutter build apk
```

iOS build, requires macOS and Xcode:

```bash
flutter build ios
```

This project is designed as a simple freelance/developer portfolio project to demonstrate Flutter UI development, forms, lists, and cross-platform mobile app structure.
